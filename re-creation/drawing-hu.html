# 插画@呼葱寻蒜

微博: link (opens new window)

无脸人像画，神韵极佳。《梦华录》实体书附增部分插画。

所谓伊人，在水一方

盈盈一水间，脉脉不得语

念去去，千里烟波，暮霭沉沉楚天阔

天街小雨润如酥，草色遥看近却无

若非群玉山头见，会向瑶台月下逢

醉后不知天在水，满船清梦压星河

柔情似水，佳期如梦

月上柳梢头，人约黄昏后

只愿君心似我心

帐里鸾凤喜合双，围红倚翠烂芬芳

美人如玉剑如虹

起舞弄清影，何似在人间

事了拂衣去，深藏身与名

飘飘乎如遗世独立，羽化而登仙
